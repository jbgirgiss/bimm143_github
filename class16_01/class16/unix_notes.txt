This is a simple nano text editor that will be available on every unix machine 
- that is why I am showing it to you. 

pwd: print working directory (where I am)
cd: change directory (move around the file system)
ls: list out files and dirs
mkdir: make a new directory (a.k.a. "folder")
rm: remove files and folders (i.e. delete, be careful since will deleted everything you are at)
nano: a basic text editor

head: show top lines of a file
tail: show bottom lines of a file
cp: copy files or dirs
mv: move or rename a file or dir
control + c: kills it
control + z: sleep/pauses the run
